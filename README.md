# Trip Service

This is a service that provides trip-related functionalities such as creating trips, estimating fares, and generating trip routes. It is built using Go and follows the microservices architecture.

## Features

- Create trips with ride fares
- Estimate fares based on trip routes
- Generate trip routes using the OSRM API
- Save ride fares and trip models to an in-memory repository

## Prerequisites

- Go version 1.16 or higher
- MongoDB (for the in-memory repository)
